Switch

() => {
  const [checked, setCheck] = React.useState(false);
  return (
    <>
      <label id="labelID" htmlFor="switchId">
        A switch with a label
      </label>
      <Switch
        checked={checked}
        className="a-class-name"
        onClick={() => setCheck(!checked)}
        aria-labelledby="labelID"
        id="switchId"
      />
    </>
  );
};

Props

NamePropTypeRequiredDefault ValueAllowed ValuesDescription
aria-labelstringfalsenull–Used to describe the purpose of the switch. To be used if there is no external label (i.e. aria-labelledby is null)
aria-labelledbystringfalsenull–A reference to a label that describes the purpose of the switch. Ignored if aria-label is provided
checkedboolfalsefalse–Whether the switch is checked or not
classNamestringfalsenull–Classes to apply to the switch container
idstringfalsenull–ID to apply to the switch container
onClickfunctrue––Function called when the switch is toggled
