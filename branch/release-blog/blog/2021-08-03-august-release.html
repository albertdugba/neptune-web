    Upgrade Guide

    1. Update @transferwise/components
    2. Import CSS from the package at the root component.
      Note: do that only if app bundles CSS (@transferwise/components defined in "dependencies"), no need add import if the package defined in "peerDependencies".
      import '@transferwise/components/build/main.css'
      Do not forget to add the import at development environments (e.g Storybook, custom demo pages).
    3. Make sure you install necessary version of React: 17.x.x or 16.14.x
    4. Check whether you need provide polyfills for some components.
      Note: only necessary for repos that bundle components
