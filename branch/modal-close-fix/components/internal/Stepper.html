Stepper

This component is used inside other components, such as FlowNavigation. Please do not use it directly.

  steps={[
    {
      label: 'One',
      onClick() {
        alert('You clicked on step 1, which triggered this function, which alerted you.');
      },
    },
    {
      label: 'Two',
      hoverLabel: (
        <>
          <div>
            <strong>Diana Jaramillo</strong>
          </div>

        </>

/>;
