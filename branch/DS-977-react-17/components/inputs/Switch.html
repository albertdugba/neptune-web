Switch

() => {
  const [checked, setCheck] = React.useState(false);
  return (
    <>
      <label id="labelID" htmlFor="switchId">
        A switch with a label
      </label>
      <Switch
        checked={checked}
        className="a-class-name"
        onClick={() => setCheck(!checked)}
        aria-labelledby="labelID"
        id="switchId"
      />
    </>
  );
};

Props

NamePropTypeRequiredDefault ValueAllowed ValuesDescription
aria-labelstringfalse––Used to describe the purpose of the switch. To be used if there is no external label (i.e. aria-labelledby is null)
aria-labelledbystringfalse––A reference to a label that describes the purpose of the switch. Ignored if aria-label is provided
checkedbooleanfalse––Whether the switch is checked or not
disabledbooleanfalse––
idstringfalse––ID to apply to the switch container
onClicksignaturetrue––Function called when the switch is toggled
