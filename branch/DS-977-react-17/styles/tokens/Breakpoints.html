Breakpoints

Experimental: Custom media values to ensure consistent breakpoints for responsive behaviour on different screen sizes.

Note: To use these properties you need to include @transferwise/neptune-css/dist/props/custom-media.css on your page. Make sure you read our guide to using custom properties if you haven't used them before.

Getting started

To use custom media custom properties you need some extra postCSS configuration:

  1. Install the postcss-custom-media plugin
  2. Add the following to your postCSS configuration (editing the path to be correct for your setup):
require('postcss-custom-media');

module.exports = {
  plugins: {
    'postcss-custom-media': {
      stage: 1,
      importFrom: './node_modules/@transferwise/neptune-css/src/props/custom-media.css',
    },
  },
};

Usage

import '@transferwise/neptune-css/dist/props/custom-media.css';

@media (--screen-md) {
  ...;
}

Breakpoint values

@custom-media --screen-xs (min-width: 480px);
@custom-media --screen-xs-max (max-width: 575px); /* --screen-sm - 1 */

@custom-media --screen-sm (min-width: 576px);
@custom-media --screen-sm-max (max-width: 767px); /* --screen-md - 1 */

@custom-media --screen-md (min-width: 768px);
@custom-media --screen-md-max (max-width: 991px); /* --screen-lg - 1 */

@custom-media --screen-lg (min-width: 992px);
@custom-media --screen-lg-max (max-width: 1199px); /* --screen-xl - 1 */

@custom-media --screen-xl (min-width: 1200px);
