Use .title-1 , .title-2 ... font styling classes for the headings. Avoid using heading elements (h1â€“h6) without any class.

Emphasis styles

Text styles can also be emphasised with a stronger priority or a different purpose.

If the style's weight is regular, then its emphasised counterpart changes to semibold. Otherwise there's no change.
